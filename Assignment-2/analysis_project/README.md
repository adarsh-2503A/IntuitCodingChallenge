# Sales Data Analysis – Java Streams Project
This project demonstrates functional programming, Java Streams, data aggregation, and CSV parsing in Java.
It reads sales data from a CSV file and performs multiple analytical operations such as:

Total revenue calculation

Revenue by region

Revenue by product

Best-selling product

Average order value

Revenue grouped by month

All analysis results are printed to the console.

Setup Instructions:

1)git clone https://github.com/adarsh-2503A/IntuitCodingChallenge.git
  cd analysis_project

2)Ensure sales.csv is in project root

3)Run the application
Open in IntelliJ → run Main.java
OR 
via terminal:
javac Main.java
java com.example.analysis.Main
